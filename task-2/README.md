# C++ Week 2 Assessment: Task 2

## Overview
This folder contains my solutions for the Week 2 C++ Assessment, focusing on core concepts like arrays, pointers, references, and structures.

## Files Included

* **`array.cpp`**: Demonstrates the use of a normal `for` loop and a `for-each` loop to print values from an integer array.
* **`pointer.cpp`**: Shows how to store a variable's address in a pointer, print its value by dereferencing, and modify the original variable via the pointer.
* **`reference.cpp`**: Demonstrates creating a reference variable to act as an alias and using it to update the original variable's value.
* **`structure.cpp`**: Defines a `Rectangle` structure with `Width` and `Breadth` attributes, instantiates it, and calculates the area.

## Compilation
These files can be compiled using any standard C++ compiler. 
Example using `g++`:
```bash
g++ pointer.cpp -o pointer
./pointer
```
